Project Overview

This project explores how violence shaped the Civil Rights Movement through the lens of memory. Using oral histories from the Library of Congress Civil Rights History Project, it focuses on how individuals remembered, responded to, and reflected on major violent events between 1955 and 1968. Rather than merely documenting what happened, this project centers how people felt, what they remembered, and how memory itself became a tool for resistance.

It is built as a digital mini-site combining narrative writing, data visualizations, mapping, and archival images to humanize and historicize the emotional landscape of civil rights violence.

Sources

Primary Dataset:
Oral history transcripts from the Library of Congress Civil Rights History Project
Selected interviewees include William G. Anderson, Jennifer Lawson, Timothy Jenkins, Junius Williams, John and Jean Rosenberg, and others.
Focused themes include violence, fear, trauma, resistance, and emotional memory.

Supplementary Sources:
Works by Patricia Sullivan, William Chafe, Yohuru Williams, Michael Edmonds, and more. Also includes historical background on events like the murder of Emmett Till, the Freedom Rides, the 16th Street Baptist Church bombing, voting-related killings, and the assassination of Dr. Martin Luther King Jr.

Processing

To analyze memory and violence across interviews:

  • Created a Google Sheet dataset tagging events, types of violence, emotions, and interviewees.
  • Used Voyant Tools to analyze frequency and context of key terms (“fear,” “jail,” “police,” etc.).
  • Applied close reading methods to extract meaningful quotes and connect them to historical context.

Presentation

This project was built using:

  • HTML/CSS in Visual Studio Code
  • GitHub Pages for public hosting
  • Quotes, narrative writing, and historical event pages
  • TimelineJS for contextual timeline
  • CARTO for mapping violent events and memory geography
  • Palladio for simple network visualizations

Technical Tutorial

This website itself functions as the technical tutorial. It shows how oral histories and memory data can be translated into narrative storytelling, interactive visuals, and structured digital archives.
